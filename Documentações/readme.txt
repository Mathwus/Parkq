*Preparação do Ambiente:

1. Download node-v8.11.1-x64
https://nodejs.org/dist/v8.11.1/node-v8.11.1-x64.msi

2. Abrir powershell, comando abaixo:
npm install -g ionic 

3. Abrir powershell na pasta "frontend" do projeto, e comando abaixo:
npm i

4. no mesmo powershell do item 3, executar:
ionic serve

*O passo 4 sobe o front. Após esse passo, só precisa executar o passo 4 no powershell pra subir o front.
